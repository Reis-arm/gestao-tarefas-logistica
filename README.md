# TechFlow Solutions - Sistema de Gerenciamento de Tarefas Logísticas

## 📋 1. Escopo Inicial do Projeto
O objetivo deste sistema é permitir que uma startup de logística gerencie suas operações diárias de transporte. A plataforma centraliza o fluxo de trabalho das entregas, triagem e manutenção de frotas, permitindo o acompanhamento em tempo real do status de cada atividade.

### Metodologia Adotada
Utilizei uma abordagem ágil baseada no Kanban para controle visual do fluxo de trabalho e respostas rápidas a mudanças, integrado ao versionamento semântico no GitHub.

---

## 🔄 2. Gestão de Mudanças (Alteração de Escopo)
**Justificativa Técnico-Operacional:** Durante as primeiras validações do MVP, a startup de logística identificou que tarefas críticas de transporte rodotransviário urgente (como cargas perecíveis ou de alto valor) estavam se misturando com tarefas operacionais comuns na mesma fila. 

**Ajuste de Escopo:** Foi necessária a inclusão emergencial de um sistema de **Classificação de Prioridades (Alta, Média, Baixa)** no CRUD de tarefas para evitar gargalos na cadeia de suprimentos e garantir que despachos de alta relevância visualmente se destaquem no quadro.
